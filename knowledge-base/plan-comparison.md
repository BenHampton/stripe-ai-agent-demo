# Plan Comparison and Features

We offer three subscription tiers: Starter, Pro, and Enterprise. All plans are
billed monthly with no long-term commitment required.

## Starter Plan — $29/month

**Best for:** Solo users and small teams getting started.

Features:
- Up to 5 users
- 10GB storage
- Core features: project management, reporting, integrations
- Email support (response within 2 business days)
- 99.5% uptime SLA

Limitations:
- No API access
- No custom integrations
- No SSO/SAML support

## Pro Plan — $79/month

**Best for:** Growing teams that need collaboration and priority support.

Features:
- Up to 25 users
- 100GB storage
- All Starter features, plus:
- API access (10,000 requests/month)
- Priority email and chat support (response within 4 hours)
- Custom integrations (up to 3)
- 99.9% uptime SLA
- Data export (CSV, JSON)

## Enterprise Plan — $299/month

**Best for:** Large organizations requiring compliance, SSO, and dedicated support.

Features:
- Unlimited users
- 1TB storage
- All Pro features, plus:
- Unlimited API access
- 24/7 dedicated support with named account manager
- SSO/SAML (Okta, Azure AD, Google Workspace)
- Custom integrations (unlimited)
- 99.99% uptime SLA with financial penalties for breaches
- SOC 2 Type II compliance reports on request
- Custom data retention policies

## Upgrading Between Plans

Upgrades take effect immediately. The customer is charged a prorated amount for the
remainder of the current billing period on the new plan, minus the unused portion
of the current plan. No action required — upgrades are automatic.

## Downgrading Between Plans

Downgrades take effect at the start of the next billing period. The customer
retains access to the current (higher) plan until the period ends.

If downgrading would exceed plan limits (e.g. 8 users on Starter which allows 5),
the customer must reduce usage before the downgrade takes effect.

## Annual Billing

Annual billing is available at a 20% discount. Annual subscriptions are
non-refundable after 14 days from the billing date. Contact support to switch
between monthly and annual billing — changes take effect at the next renewal.
